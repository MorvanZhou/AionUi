# 🏕️ TCampos Skill

访问 [tcampos.qq.com](https://tcampos.qq.com) 平台数据的 AI 技能。

## 💬 直接对话即可使用

| 你可以这样问                   | AI 会帮你做什么                       |
| ------------------------------ | ------------------------------------- |
| "查一下有哪些营地"             | 列出所有营地班级                      |
| "找到 MINI鹅冬季营 的报名情况" | 搜索营地并显示报名统计                |
| "导出这个营地的用户名单"       | 下载营地用户 Excel 到 `./downloads/`  |
| "看看入营协议签署了多少人"     | 查询任务完成进度                      |
| "给必修任务未完成的用户发通知" | 筛选用户并生成通知模版到 `./outputs/` |
| "查询用户 张三 的详细信息"     | 搜索用户档案                          |

> 🔐 **首次使用**：AI 会询问手机号和密码登录，之后自动管理 Token。

---

## 🎯 核心能力

| 类别         | 能力                                               |
| ------------ | -------------------------------------------------- |
| **营地管理** | 查询营地列表、报名筛选流程、导出报名/笔试/面试数据 |
| **用户管理** | 导出营地用户、查询用户档案、查询用户事件记录       |
| **任务管理** | 查询任务列表、导出任务完成记录、下载附件           |
| **主题营**   | 查询主题营列表、查看关联营地                       |
| **数据分析** | Excel 跨表合并、筛选统计、完成度分析               |
| **通知发送** | 筛选目标用户、生成通知模版                         |

---

## 📁 文件输出目录

| 类型     | 目录           | 说明                  |
| -------- | -------------- | --------------------- |
| 下载数据 | `./downloads/` | API 导出的 Excel 文件 |
| 生成数据 | `./outputs/`   | 分析处理后生成的文件  |

---

## 🚀 使用示例

**查找营地并导出用户**

```
用户：帮我找到 "2026腾讯MINI鹅冬季营" 并导出用户名单
AI：搜索营地 → 导出用户表到 ./downloads/
```

**追踪任务完成情况**

```
用户：看看入营协议签署了多少人，把没签的人联系方式导出来
AI：查询任务 → 统计完成人数 → 导出未完成者联系方式
```

**批量发送通知**

```
用户：给必修任务没完成的用户发通知
AI：导出用户表 → 筛选未完成用户 → 生成通知模版到 ./outputs/
```
